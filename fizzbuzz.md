## fizzbuzz(スペルは知らない)

という名前のゲームがあります.

適当にwikiをみて詳細を知ってください.

ちょっと作ってみてください.
___

### ルール

0から100までの数字を一つずつ挙げる.

挙げた数字が,

3で割り切れるなら
```
fizz
```
5で割り切れるなら
```
buzz
```
どちらでも割り切れるなら
```
fizzbuzz
```

と表示する.

### 補足
 * 0については,適当に処理してください.
 * あまりが計算したければ%を使ってください.

### 例
```
for var in range(0,101):
  if var%3==0:
    ...
```
